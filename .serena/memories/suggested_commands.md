# 常用命令

## 开发

- `pnpm dev` - 启动开发服务器
- `pnpm build` - 构建生产版本
- `pnpm preview` - 预览生产构建

## 代码质量

- `pnpm lint` - 运行 ESLint 检查
- `pnpm format` - 格式化代码 (Prettier)
- `pnpm format:check` - 检查格式 (不修改文件)

## 测试

- `pnpm test` - 运行测试 (watch 模式)
- `pnpm test:run` - 运行测试一次
- `pnpm test:ui` - 启动测试 UI
- `pnpm test:coverage` - 生成覆盖率报告

## Storybook

- `pnpm storybook` - 启动 Storybook
- `pnpm build-storybook` - 构建 Storybook

## 系统命令 (macOS)

- `ls`, `cd`, `grep`, `find` - 标准 Unix 命令
- `git` - 版本控制
